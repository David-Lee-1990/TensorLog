Various artificial learning tasks inspired by experiments in:

Dries, Anton, et al. "ProbLog2: Probabilistic logic programming."
Joint European Conference on Machine Learning and Knowledge Discovery
in Databases. Springer, Cham, 2015.

expt.py, testexpt.py, tfexpt.py are for automated tests (using a small
graph based on a the cora citation data).

scaleup/* is support for scalability experiments on different-sized
artificially generated graphs in JAIR submission
